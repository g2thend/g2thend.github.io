关于目录及文件的权限

/etc/fstab复制到/data/dir，⾄少需要什么权限？

cp命令 有执⾏权限 /etc/fstab 读权限 /data/dir 写
权限和执⾏权限

当⽤户docker对/testdir ⽬录⽆执⾏权限时，意味着⽆法做哪些操作？

  • 无法进入目录
  • 可以用ls查看列表，但是无法查看文件属性类型
  • 就算对目录中的文件有rw权限，也无法查看和写入>>也不行

当⽤户mongodb对/testdir ⽬录⽆读权限时，意味着⽆法做哪些操作？#

  • 可以进入目录，无法查看文件列表，可以查看ll file详细信息
  • 可以执行目录内程序，cat file可以查看文件内容
  • 如果对文件有w权限，还可以通过>>写入文件

当⽤户redis 对/testdir ⽬录⽆写权限时，该⽬录下的只读⽂件file1是否可修改和删除？

  • 不能在该目录下删除和创建文件。

当⽤户zabbix对/testdir ⽬录有写和执⾏权限时，该⽬录下的只读⽂件file1是否可修改和删除？#

  • 不能改、可删除。






grep正则表达式常用字符（自用）
常用选项

  • -c 显示匹配到的行的数目，不显示内容
  • -n 相对行号
  • -w 表达式当作词来寻找，类似\< > 界定符
    扩展的正则表达式元字符(egrep)
  • + : 一个或多个前字符
  • \? : 零个或一个
  • a|b : a 或 b
  • () : 字符组
  • [:alnum:]　数字字母
  • [:alpha:] 字母
  • [:ctrl:] 控制字符
  • [:digit:]
  • [:graph:] 非空字符（不包含空格，控制字符）
  • [:lower:]
  • [:print:] 与[:graph:]类似,但包含空格字符
  • [:punct:] 标点字符
  • [:upper:]
  • \w 所有字母与数字称之为词字符
  • \W 所有字母与数字之外的字符，非词 字符
    例子
  • ‘love|hate’
  • love(able|ly)(ov)+




    登录的基本流程
    PID为1的init进程(systemd),从文件inittab 读取指令，或者派生出getty进程，这些进程打开终端接口，提供标准输入输出，并显示登录提示符，接下来执行/bin/login，其依次执行：提示用户输入口令，加密验证口令，设置初始环境，启动用户的登录shell，登录shell先查找系统/etc/profile,并执行命令，然后再用户的主目录下查找.profile 并执行，并显示默认的命令提示符。

doti(.) 命令

内置命令,以脚本名为参数，在当前目录下执行，并不启动新shell，成为当前环境的一部分。

exec < file
命令exec 将file作为标准输入，读取并执行，顶替了当前的shell运行，当命令执行完毕，shell随之退出，即当前shell被销毁。

  • exec 3> file1
  • who >& 3
  • date >& 3
  • exec 3>&-
  • exec 3<& file
  • cat <&3
  • exec <&3-
